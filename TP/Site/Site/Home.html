ELEMENTOS AQUI!!!!!

HOME: se estiver logado mostra, as materias escolhidas e as proximas materias no decorrer do curso.

CADASTRO: onde o usuario vai realizar o cadastro das materias, marcando as suas prioridades para cada uma.

HORÁRIOS: mostra a grade horaria do usuario que foi feita no item CADASTRO.

CONFIGURAÇÕES: (talvez opcional) o usuario consegue excluir materias que ele fez o cadastro,

CONTA: se o usuario nao estiver logado, aparece o login para ele. Caso estiver logado, aparece a opção de deslogar.
